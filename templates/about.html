Stock Market Prediction

Using Data-Driven Ensemble ML Methods

Project Overview

This project focuses on predicting stock market trends using a combination of machine learning models. The ensemble approach leverages multiple prediction techniques to improve accuracy and reliability. The system fetches stock data, processes it, and applies various models to forecast future prices.

Algorithms Used

  • ARIMA - Time series forecasting model.
  • LSTM (Long Short-Term Memory) - Deep learning model for sequential data.
  • Linear Regression - Machine learning model for stock prediction.

Libraries Used

  • alpha_vantage - Fetches stock data from Alpha Vantage.
  • pandas & numpy - Data handling and manipulation.
  • mean_squared_error - Measures prediction accuracy.
  • matplotlib - Plots and visualizes stock trends.
  • math, random - Performs mathematical operations.
  • datetime - Handles date and time functionalities.
  • yfinance - Fetches stock data from Yahoo Finance.
  • tweepy - Twitter API access (possibly for sentiment analysis).
  • preprocessor, re, TextBlob - Text preprocessing for sentiment analysis.
  • nltk.download('punkt') - Downloads a tokenizer for text processing.
